## 🚀 Featured Projects

### 🏗️ AI Auto Code Builder
**Tech Stack:** Python, LangGraph, LLM (OpenAI/Claude/Gemini), LangChain, Tavily Search API

**Description:** An autonomous multi-agent system designed to handle the end-to-end software development lifecycle. Built using **LangGraph**, the system employs specialized agents for coding, reviewing, and testing. It utilizes a stateful graph-based workflow to manage complex logic, allowing agents to collaborate, iterate on code based on feedback, and perform web searches to solve programming challenges. This project demonstrates the power of **Agentic Workflows** in automating technical tasks with high precision.

🔗 **Repository:** [https://github.com/srinitish/AI-Auto-Code-Builder-using-LangGraph-and-LLM-Agents](https://github.com/srinitish/AI-Auto-Code-Builder-using-LangGraph-and-LLM-Agents)

### 🍽️ DishDraft AI (FlavorGen AI)
**Tech Stack:** Python, LangChain, Google Gemini API, Streamlit, Prompt Engineering

**Description:** A smart creative assistant for the culinary industry that automates restaurant branding and menu design. By leveraging Generative AI, the application generates catchy, niche-specific restaurant names and crafts detailed, themed menus based on user-inputted cuisines. This project highlights the use of **Chain-of-Thought prompting** and **LLM orchestration** to transform simple ideas into structured, professional business assets through a seamless Streamlit interface.

🔗 **Repository:** [https://github.com/srinitish/DishDraft-AI-Smart-Restaurant-Naming-Menu-Design](https://github.com/srinitish/DishDraft-AI-Smart-Restaurant-Naming-Menu-Design)
